Backend és API fejlesztés

Mivel foglalkozom?

Gépészmérnökként dolgozom egy autóipari vállalatnál. 2021-ben kezdtem el a Pythont tanulni. Egy év tanulás után kisebb projekteket készítettem különböző könyvtárak alkalmazásával. Portfólió projektként ezt a bemutatkozó weboldalt készítettem, HTML és CSS kódból felépítve, Bootstrap és JavaScript nélkül. Pythonban a Backend és API fejlesztést tanulom és erre is szeretnék specializálódni. A megszerzett tudást szeretném egy fejlesztői cégnél vagy szabadúszóként használni.
