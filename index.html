Mivel foglalkozom?

Gépészmérnökként dolgozom egy autóipari vállalatnál. 2021-ben kezdtem el a Pythont tanulni. Egy év tanulás után kisebb projekteket készítettem különböző modulok alkalmazásával. Jelenleg webfejlesztést tanulok a Frontend megértése végett. HTML+CSS Projektként ezt a bemutatkozó weboldalt készítettem, tisztán HTML és CSS kódból felépítve, Bootstrap és JavaScript nélkül. Erősségem Pythonban az adat analízis. Célom a Frontend és Backendben való fejlődés, API-k készítése. A megszerzett tudást szeretném egy fejlesztői cégnél használni, főleg Python Backend vonalon, de a Frontend vonal is érdekel.
